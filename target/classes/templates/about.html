Gute Idee! (ˈɡuːtə iˈdeː) – good idea!

An app to capture your flashes of genius (they always come at the weirdest times). Organize them, get others inspired, and then get them done.

My name is Olaf, I'm based in Berlin, Germany, but I've been traveling the world as a freelance DoP / cameraman. I've always been a bit of a tech aficionado as well, so over time I got curious about how it all works on a deeper level. I've designed a few private websites but never really programmed (except for a RasPi based photo booth maybe), however having control over what is going on seems very appealing to me and so I decided to dig deeper, which led to attending a 6-month full-stack Java Web App Bootcamp over at CodersCampus. This little Gute Idee! app is the final project I coded from scratch for said bootcamp.

Currently I'm studying computer science part-time while working on filming projects, writing for an online magazine and of course honing my Java skills.

Technologies used include Java 11, MySQL database, Hibernate, Thymeleaf, Javascript, Bootstrap CSS and of course a fair bit of HTML.

You can check out the code for this project, and others, over at GitHub.
